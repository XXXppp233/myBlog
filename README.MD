# Extended Features

This blog supports extended Markdown syntax and HTML embedding, similar to Cloudflare Docs.

## Video Embeds

You can embed YouTube videos or other iframes directly.

::: embed

<iframe width="100%" height="400" src="https://www.youtube.com/embed/dQw4w9WgXcQ" title="YouTube video player" frameborder="0" allow="accelerometer; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" allowfullscreen></iframe>
:::

## Custom Alerts

Use HTML for custom alerts.

::: alert
  **Alert:** This is a custom alert box style.
:::

::: note
  **Note:** This is a custom note box style.
:::
